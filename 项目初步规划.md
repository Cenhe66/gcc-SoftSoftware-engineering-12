# 智能停车管理系统 —— 项目初步规划

> 基于《智能停车管理系统完整创新功能汇总》提炼，用于指导后续开发实施。

---

## 一、核心功能模块梳理

系统拆分为 **7大业务模块 + 3大技术支撑层**：

| 模块 | 核心功能 | 技术重点 |
|------|---------|---------|
| **智能引导** | 车位热力图、离场预判、AR寻车、个性化推荐 | WebSocket实时推送、Canvas/AR、陀螺仪 |
| **预约管理** | 长周期预约、超时管控、地锁联动 | 定时任务、Redis过期事件 |
| **共享车位** | 私有车位共享、智能地锁、分账管理 | 状态机、账单计算 |
| **健康检测** | 车窗/车灯/胎压异常检测 | 规则引擎、模拟数据 |
| **匿名挪车** | 扫码挪车、快捷消息、WebSocket通讯 | 用户脱敏、实时消息 |
| **缴费与访客** | 无感支付、访客授权 | 支付沙箱、二维码/白名单 |
| **管理后台** | 数据大屏、车流统计、收益看板 | ECharts、Vue + Element UI |

## 二、系统架构设计（三层 + 两层支撑）

```
+-----------------------------------------+
|           展示层 (Presentation)          |
|  +-------------+    +-----------------+ |
|  |  微信小程序  |    |  Vue管理后台     | |

|  +-------------+    +-----------------+ |
+-----------------------------------------+
|           业务层 (Business)              |
|         SpringBoot 单体服务              |

|           数据层 (Data)                  |
|     MySQL 8.0  +  Redis 6.x             |
|   结构化数据存储    缓存/状态/会话        |
+-----------------------------------------+
```

## 七、关键风险与答辩准备点

| 功能 | 真实场景难点 | 毕设规避方案 |
|------|------------|------------|
| AR寻车 | 室内定位精度 | 预标定坐标 + 陀螺仪方向模拟 |
| 离场预判 | 需要大量历史数据训练AI | 规则引擎（停留时长+预约时间）模拟 |
| 无感支付 | 需微信支付商户资质 | 沙箱环境/模拟支付流水 |
| 地锁联动 | 需硬件IoT对接 | 软件状态机模拟 + WebSocket指令 |
| 视觉识别 | 需AI模型与摄像头 | 模拟传感器数据 + 规则触发 |
