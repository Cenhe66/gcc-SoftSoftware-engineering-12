# Vue管理后台数据大屏开发计划

> 基于 Vue 3 + Element Plus + ECharts 开发的数据可视化大屏

---

## 一、技术选型

| 技术项 | 选择 | 说明 |
|--------|------|------|
| 框架 | Vue 3 | 组合式API，更好的TypeScript支持 |
| UI组件库 | Element Plus | 成熟的Vue 3组件库 |
| 图表库 | ECharts 5 | 强大的数据可视化能力 |
| 状态管理 | Pinia | Vue 3官方推荐 |
| 路由 | Vue Router 4 | Vue 3配套路由 |
| HTTP请求 | Axios | 成熟的HTTP客户端 |
| 构建工具 | Vite | 快速的开发服务器 |

## 五、API接口对接清单

| 功能 | 接口路径 | 说明 |
|------|---------|------|
| 统计数据 | GET /api/dashboard/stats | 今日停车、收益、空闲车位等 |
| 车流趋势 | GET /api/dashboard/traffic | 24小时车流数据 |
| 收益趋势 | GET /api/dashboard/revenue | 近7天/30天收益 |
| 热力图 | GET /api/dashboard/heatmap | 车位使用率热力图 |
| 停车场排行 | GET /api/dashboard/ranking | 收益/使用率排行 |
| 告警信息 | GET /api/dashboard/alerts | 系统告警列表 |
| WebSocket | ws://localhost:8080/ws/admin | 实时数据推送 |

## 六、设计规范

### 主题色
- 主色：`#409EFF`（Element Plus蓝色）
- 背景色：`#0a1929`（深色科技风）
- 卡片背景：`rgba(255, 255, 255, 0.05)`
- 文字主色：`#ffffff`
- 文字次色：`rgba(255, 255, 255, 0.7)`

### 字体规范
- 大屏标题：32px
- 卡片标题：18px
- 统计数据：48px（加粗）
- 正文：14px

### 间距规范
- 卡片间距：20px
- 内边距：20px
- 圆角：8px
