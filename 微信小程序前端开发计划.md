# 智能停车管理系统 —— 微信小程序前端开发计划

> 基于微信原生小程序开发（WXML + WXSS + JS），对接已完成的后端API

---

## 一、技术选型

| 技术项 | 选择 | 说明 |
|--------|------|------|
| 框架 | 微信原生小程序 | 毕设推荐，文档完善，无需额外学习成本 |
| UI组件 | Vant Weapp | 轻量、美观的微信小程序UI组件库 |
| 状态管理 | 原生Storage + 全局数据 | 小程序原生支持，足够应对毕设规模 |
| 地图服务 | 腾讯位置服务 | 与微信生态深度整合 |
| 网络请求 | 封装wx.request | 统一拦截、错误处理 |

## 四、核心页面详细设计

### 4.1 首页 (pages/index)

**功能点：**
- 顶部搜索栏（搜索停车场名称/地址）
- 轮播图（广告/活动）
- 快捷入口图标（预约、共享、挪车、AR寻车）
- 附近停车场列表卡片
- 底部Tab导航

**接口调用：**
- `GET /api/parking-lot/list` - 获取停车场列表
- `GET /api/parking-lot/detail/{id}` - 获取详情

### 4.2 停车场详情 (pages/parking-detail)

**功能点：**
- 停车场基本信息（名称、地址、总车位、空闲数）
- 收费标准展示
- 车位分布简化图（楼层/区域）
- 导航按钮（调起腾讯地图）
- 预约入口

**接口调用：**
- `GET /api/parking-lot/detail/{id}`
- `GET /api/parking-space/list?lotId={id}` - 车位列表

### 4.3 预约页面 (pages/reservation)

**功能点：**
- 日期时间选择器
- 预计时长计算
- 预估费用显示
- 提交预约

**接口调用：**
- `POST /api/reservation/create` - 创建预约

### 4.4 AR寻车 (pages/ar-find-car)

**功能点：**
- camera组件预览
- 陀螺仪方向感应
- 车位方向箭头指引
- 距离显示

**技术要点：**
- 使用`wx.startGyroscope`监听陀螺仪
- 使用`wx.createCameraContext`控制相机
- 根据车位坐标计算相对方向

### 4.5 匿名挪车 (pages/move-car)

**功能点：**
- 扫码入口（模拟扫码）
- 车辆信息展示（脱敏）
- 快捷消息选择
- 发送挪车请求

**接口调用：**
- `POST /api/move-car/request` - 发起请求
- `POST /api/move-car/message/send` - 发送消息
- `GET /api/move-car/message/list/{requestId}` - 消息列表

## 六、WebSocket实时通信

**连接地址：** `ws://localhost:8080/ws/parking`

**消息类型：**

| 消息类型 | 说明 | 处理页面 |
|---------|------|---------|
| HEATMAP_UPDATE | 热力图数据更新 | index |
| SPACE_STATUS_CHANGE | 车位状态变更 | parking-detail |
| MOVE_CAR_MESSAGE | 挪车新消息 | move-car-message |
| RESERVATION_REMIND | 预约到期提醒 | reservation-list |
| PAYMENT_SUCCESS | 支付成功通知 | payment |
